# Mini StackOverflow - Frontend (React + Tailwind)

This is a starter front-end project (Vite + React + Tailwind) for a Mini StackOverflow-style application.

## Quick start

1. Install dependencies
```
npm install
```

2. Start dev server
```
npm run dev
```

3. Tailwind is preconfigured. If your backend runs on localhost:8000 the example API client will call it.

## Project structure

- src/
  - App.jsx (routes)
  - main.jsx
  - index.css
  - Pages/
  - Components/common/
  - lib/api.js

Customize pages and components to connect to your backend.
